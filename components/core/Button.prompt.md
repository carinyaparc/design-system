Pill button for primary actions; use one primary per view, outline/ghost for secondary paths.

```jsx
<Button variant="primary" size="md">Learn our story</Button>
<Button variant="outline">Plan a visit</Button>
<Button variant="ghost">Read more →</Button>
```

Variants: primary (eucalypt), secondary (kangaroo gold), outline, ghost (bracken text). Sizes sm/md/lg. `block` for full width.
