# Website UI kit — Carinya Parc

A high-fidelity recreation of the public marketing site home page in the new brand.
Recreated from `apps/site/src/app/(www)/page.tsx` (Next.js 16 + Tailwind CSS 4 + Base UI).

- `index.html` — the home page: sticky nav, image hero, mission trio, story split, experience band, journal cards, footer.

Composes the same primitives as `components/` (buttons, tags, cards) and reads all colour/type/spacing from `styles.css`. Imagery lives in `/assets`.
